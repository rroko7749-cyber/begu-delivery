# 🚀 Самый простой способ получить APK

## Используем Expo Application Services (бесплатно)

### Шаг 1: Установи EAS CLI

Открой командную строку:

```bash
npm install -g eas-cli
```

### Шаг 2: Создай бесплатный аккаунт Expo

```bash
eas login
```

Или зарегистрируйся на https://expo.dev (бесплатно)

### Шаг 3: Настрой проект

```bash
cd C:\bat\delivery-app
eas build:configure
```

### Шаг 4: Собери APK

```bash
eas build --platform android --profile preview
```

⏱️ Сборка займёт 10-15 минут на серверах Expo.

После завершения получишь ссылку для скачивания APK!

---

## Альтернатива: Я соберу за тебя

Если не хочешь возиться с регистрацией, могу:

1. Создать простую HTML5 версию (работает в браузере телефона)
2. Или дать инструкцию как собрать APK через Android Studio

**Что выбираешь?**
- Вариант 1: Сам соберёшь через EAS (10 мин)
- Вариант 2: HTML5 версия (работает в браузере, без установки)
- Вариант 3: Полная инструкция по Android Studio (60 мин настройки)
