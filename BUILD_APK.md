# Как собрать APK файл

## Вариант 1: Через EAS Build (Рекомендуется)

### Шаг 1: Установи EAS CLI
```bash
npm install -g eas-cli
```

### Шаг 2: Войди в Expo аккаунт
```bash
eas login
```

Если нет аккаунта - создай на https://expo.dev

### Шаг 3: Собери APK
```bash
cd C:\bat\delivery-app
eas build --platform android --profile preview
```

Это займёт 10-15 минут. APK соберётся на серверах Expo.

### Шаг 4: Скачай APK
После сборки получишь ссылку на скачивание APK файла.

---

## Вариант 2: Локальная сборка (без интернета)

Если не хочешь использовать облако Expo, могу настроить локальную сборку через Android Studio.

**Что нужно:**
1. Установить Android Studio (~3GB)
2. Настроить SDK
3. Собрать APK локально

Это займёт больше времени, но APK будет собран на твоём компьютере.

---

## Вариант 3: Упрощённый APK (без Expo)

Создам простую версию приложения на чистом React Native и соберу APK.

**Плюсы:**
- Меньше размер APK
- Полный контроль

**Минусы:**
- Дольше настраивать
- Нужен Android Studio

---

## Какой вариант выбрать?

**Для быстрого теста:** Вариант 1 (EAS Build)
**Для продакшена:** Вариант 2 или 3
