# 🚀 Быстрый старт - Запуск на телефоне за 5 минут

## 1️⃣ Установи Expo Go на телефон

📱 Скачай приложение **Expo Go**:
- Android: [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent)
- iPhone: [App Store](https://apps.apple.com/app/expo-go/id982107779)

## 2️⃣ Установи зависимости

Открой командную строку (Win+R → `cmd`) и выполни:

```bash
cd C:\bat\delivery-app
npm install
```

⏱️ Это займёт 5-10 минут (скачает все библиотеки)

## 3️⃣ Запусти проект

```bash
npm start
```

Откроется браузер с QR-кодом.

## 4️⃣ Открой на телефоне

**Android:**
1. Открой Expo Go
2. Нажми "Scan QR code"
3. Наведи камеру на QR-код
4. Готово! 🎉

**iPhone:**
1. Открой камеру
2. Наведи на QR-код
3. Нажми уведомление
4. Готово! 🎉

---

## ⚠️ Важно

- Телефон и компьютер должны быть в **одной Wi-Fi сети**
- Не закрывай командную строку
- Изменения в коде сразу видны на телефоне

## 🐛 Проблемы?

**QR-код не работает:**
```bash
npm start -- --tunnel
```

**Ошибки при установке:**
```bash
npm cache clean --force
npm install
```
