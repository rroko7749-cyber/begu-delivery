# Как запустить на телефоне через Expo

## Шаг 1: Установи Expo Go на телефон

📱 **На Android:**
- Открой Google Play
- Найди "Expo Go"
- Установи приложение

📱 **На iPhone:**
- Открой App Store
- Найди "Expo Go"
- Установи приложение

## Шаг 2: Установи зависимости на компьютере

Открой командную строку в папке `C:\bat\delivery-app` и выполни:

```bash
npm install
```

Это займёт 5-10 минут.

## Шаг 3: Запусти проект

```bash
npm start
```

Откроется окно с QR-кодом.

## Шаг 4: Открой на телефоне

**Android:**
1. Открой приложение Expo Go
2. Нажми "Scan QR code"
3. Наведи камеру на QR-код на экране компьютера
4. Приложение загрузится на телефон!

**iPhone:**
1. Открой стандартную камеру
2. Наведи на QR-код
3. Нажми на уведомление "Открыть в Expo Go"
4. Приложение загрузится!

## Важно!

- Телефон и компьютер должны быть в **одной Wi-Fi сети**
- Не закрывай окно командной строки пока работаешь с приложением
- Изменения в коде сразу отобразятся на телефоне

## Если не работает

**QR-код не сканируется:**
- Убедись что телефон и компьютер в одной сети
- Попробуй ввести адрес вручную (он показан под QR-кодом)

**Ошибка подключения:**
- Проверь что не блокирует файрвол
- Попробуй запустить через туннель: `npm start -- --tunnel`

## Готово!

Теперь можешь тестировать приложение прямо на телефоне. Все изменения в коде будут сразу видны на экране.
