## 📋 Что нужно сделать

### Этап 1: Установка и настройка (1-2 дня)

1. **Установить React Native окружение**
   ```bash
   # Установить Node.js (если нет)
   # Скачать с https://nodejs.org
   
   # Установить Android Studio
   # Скачать с https://developer.android.com/studio
   
   # Установить зависимости проекта
   cd delivery-app
   npm install
   ```

2. **Настроить Firebase** (следуй `FIREBASE_SETUP.md`)
   - Создать проект
   - Настроить Authentication
   - Настроить Firestore
   - Скопировать конфигурацию в `src/services/firebase.js`

3. **Запустить приложение**
   ```bash
   # Запустить на эмуляторе Android
   npm run android
   ```

### Этап 2: Интеграция Firebase (3-5 дней)

**Что добавить:**

1. **Авторизация** (`src/screens/common/LoginScreen.js`)
   - Вход по номеру телефона
   - SMS-код подтверждения
   - Регистрация пользователя

2. **Создание заказов** (`src/screens/client/CreateOrderScreen.js`)
   - Сохранение заказа в Firestore
   - Уведомление курьеров

3. **Список заказов** (`src/screens/courier/AvailableOrdersScreen.js`)
   - Загрузка заказов из Firestore
   - Фильтр по расстоянию
   - Обновление в реальном времени

4. **Активная доставка** (`src/screens/courier/ActiveDeliveryScreen.js`)
   - Обновление статуса заказа
   - Расчёт заработка
   - Сохранение в earnings

### Этап 3: Карты и геолокация (2-3 дня)

1. **Установить библиотеки**
   ```bash
   npm install react-native-maps
   npm install @react-native-community/geolocation
   ```

2. **Добавить карты**
   - Яндекс.Карты API (бесплатно до 25k запросов/день)
   - Показ маршрута
   - Трекинг курьера в реальном времени

3. **Геолокация**
   - Определение местоположения курьера
   - Расчёт расстояния
   - Навигация к точкам

### Этап 4: Чат и уведомления (2-3 дня)

1. **Чат клиент-курьер**
   - Firebase Realtime Database
   - Простой текстовый чат

2. **Push-уведомления**
   - Firebase Cloud Messaging
   - Уведомления о новых заказах
   - Уведомления о статусе доставки

### Этап 5: Тестирование (3-5 дней)

1. **Внутреннее тестирование**
   - Ты + друзья
   - Проверка всех функций
   - Исправление багов

2. **Закрытое бета-тестирование**
   - 10-20 человек
   - Реальные заказы
   - Сбор обратной связи

### Этап 6: Запуск MVP (1-2 недели)

1. **Подготовка к публикации**
   - Иконка приложения
   - Скриншоты для Google Play
   - Описание приложения
   - Политика конфиденциальности

2. **Публикация в Google Play**
   - Создать аккаунт разработчика ($25 один раз)
   - Загрузить APK
   - Заполнить информацию
   - Отправить на модерацию (1-3 дня)

3. **Мягкий запуск**
   - Запуск в 1-2 городах
   - Привлечение первых курьеров
   - Привлечение первых клиентов
