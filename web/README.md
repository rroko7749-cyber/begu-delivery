# 📱 Веб-версия приложения (работает как нативное!)

## Что это?

Это Progressive Web App (PWA) - веб-приложение которое работает как нативное:
- ✅ Устанавливается на главный экран
- ✅ Работает без интернета
- ✅ Выглядит как обычное приложение
- ✅ Не требует Google Play

## Как установить на телефон?

### Android (Chrome):

1. Открой на телефоне файл `web/index.html` через любой веб-сервер
2. Или загрузи на хостинг (например, GitHub Pages - бесплатно)
3. Открой в Chrome
4. Нажми меню (⋮) → "Установить приложение"
5. Готово! Иконка появится на главном экране

### iPhone (Safari):

1. Открой файл в Safari
2. Нажми кнопку "Поделиться" 
3. Выбери "На экран Домой"
4. Готово!

## Быстрый запуск (локально):

```bash
cd C:\bat\delivery-app\web
python -m http.server 8000
```

Затем открой на телефоне: `http://IP_КОМПЬЮТЕРА:8000`

(Узнай IP: `ipconfig` в командной строке, найди IPv4)

## Или загрузи на хостинг:

### GitHub Pages (бесплатно):

1. Создай репозиторий на GitHub
2. Загрузи папку `web`
3. Включи GitHub Pages в настройках
4. Получишь ссылку типа `https://username.github.io/deliverypro`

### Netlify (бесплатно):

1. Зайди на netlify.com
2. Перетащи папку `web`
3. Получишь ссылку

## Преимущества PWA:

- Не нужен Google Play
- Работает на Android и iPhone
- Обновляется автоматически
- Легко распространять (просто ссылка)

## Недостатки:

- Нет доступа к некоторым функциям телефона
- Нужен интернет для первой загрузки
